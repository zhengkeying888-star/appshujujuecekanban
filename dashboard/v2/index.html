指标口径说明：
• CTR = 点击UV / 曝光UV
• CTCVR（点击转化率/领课转化率）= 线索数 / 点击UV
• 线索生成率 = 线索数 / 曝光UV
• 好友率 = 加好友数 / 线索数
• 到课率 = 到课数 / 加好友数
• 完课率 = 完课数 / 到课数
• 完课→首单转化率 = 首单数 / 完课数
• 线索→首单转化率 = 首单数 / 线索数
• 单线索产出(LTV) = GMV / 线索数
• 单曝光产出 = GMV / 曝光UV
• 单点击产出 = GMV / 点击UV
