# APP 线索广告位复盘月报 — 产品需求文档（PRD）v3.1

> 文档版本：v3.2
> 更新日期：2026-05-13
> 目标用户：运营团队、管理层、产品侧
> 数据周期：2026 年 3–4 月

## 1. 项目背景与目标

基于 `APP线索广告位拆解 3-4月更新版.xlsx`（约 3.6 万行、37 列明细数据）、`3-4月月活人数.xlsx`（用户等级月活数据）及 `APP品类流量结构.csv`（品类级月度汇总），构建可本地运行的单文件 HTML BI 看板，并支持一键导出为独立 HTML 月报。

核心诉求：
1. 快速掌握月度线索、GMV、转化率核心指标及环比变化
2. 定位 15 个核心资源位的效率波动与衰退原因
3. 分析素材卖点效果（提取 `【】` 内关键词）
4. 三维/四维交叉分析（品类 × 价格带 × 人群 × 卖点）找出最优组合
5. 资源位与品类的交叉效率评估（热力图 + 推荐矩阵 + 逐资源位明细）
6. 品类流量结构变化趋势（区分正式品/孵化品）
7. **新增 v3.0**：价格带全局占比分布（区分正式品/孵化品）
8. **新增 v3.0**：资源位 × 品类类型效率对比（正式品 vs 孵化品）
9. **新增 v3.2**：月活人数（MAU）与线索生成率纳入核心 KPI，支持按用户等级拆分月活趋势

## 2. 数据源与关键字段

### 2.1 主数据源

**文件**：`APP线索广告位拆解 3-4月更新版.xlsx`
**规模**：35,558 行（过滤合计后），37 列

| 字段 | 说明 | 用途 |
|-----|------|------|
| `stat_month` | 统计月份（2026-03 / 2026-04 / 合计） | 月份切分与过滤 |
| `tag_level_1` | 投放位置名称（原 `广告资源位`） | 资源位效率分析、热力图 Y 轴 |
| `category_name` | 品类名称 | 品类分析、热力图 X 轴 |
| `camp_name` | 活动/课程名称 | 卖点分析回退数据源（原 `广告位素材` 已移除） |
| `sku_price` | 课程价格 | 价格带映射（0元/1.1元/3.9元/其他） |
| `线索数` | 投放获客数 | 核心指标 |
| `首单数` | 成交订单数 | 转化率分子 |
| `首单流水` | 成交金额（GMV） | 核心指标 |
| `LTV` | 用户生命周期价值 | LTV 均值计算 |
| `转化率` | 行级转化率（= 首单数/线索数） | **仅作验证，聚合时严禁直接 AVG** |

> **注**：新文件已移除 `广告资源位`、`广告位素材`、`面向人群` 三列。`tag_level_1` 等效替代 `广告资源位`；`camp_name` 替代 `广告位素材` 用于卖点提取；`面向人群` 自动填充为 `'全部用户'`。

### 2.2 月活数据源（v3.2 新增）

**文件**：`3-4月月活人数.xlsx`
**Sheet**：`用户等级月活人数`
**规模**：22 行（11 个用户等级 × 2 个月）

| 字段 | 说明 | 用途 |
|-----|------|------|
| `月份` | 2026-03 / 2026-04 | 月份切分 |
| `用户等级` | 0–10 | 等级拆分维度 |
| `月活人数` | 该等级当月活跃用户数 | MAU 计算与趋势图 |
| `占比` | 该等级占当月总 MAU 比例 | 辅助验证 |

### 2.2 品类流量结构数据源

**文件**：`APP品类流量结构.csv`
**格式**：左右并排双表，左侧为 4 月数据，右侧为 3 月数据
**关键列**：`品类区别`（正式品 / 孵化品）

## 4. 指标口径定义

### 4.1 核心指标计算公式

| 指标名称 | 计算公式 | 数据来源字段 | 说明 |
|---------|---------|------------|------|
| **月度线索数** | `SUM(线索数)` | `线索数` | 按统计月份汇总 |
| **GMV（首单流水）** | `SUM(首单流水)` | `首单流水` | 单位：元 |
| **转化率** | `SUM(首单数) / SUM(线索数)` | `首单数`, `线索数` | **严禁直接 AVG 原始 `转化率` 字段** |
| **LTV 均值** | `AVG(LTV)` | `LTV` | 算术平均，保留 2 位小数 |
| **月活人数 (MAU)** | `SUM(月活人数)` | `月活人数` | 按月份汇总所有用户等级的月活 |
| **线索生成率** | `SUM(线索数) / SUM(月活人数) × 100%` | `线索数`, `月活人数` | 反映流量池的线索获取效率，保留 2 位小数 |
| **环比** | `(本月值 - 上月值) / 上月值 × 100%` | — | 上月值为 0 时显示「—」 |
| **环比绝对值** | `本月值 - 上月值` | — | KPI 卡片下方辅助说明 |
| **综合得分** | `CVR_norm×40% + GMV_norm×35% + Leads_norm×25%` | — | Min-Max 归一化到 0-100 后加权，仅对 4 月线索 ≥ 20 的组合计算 |

### 4.2 价格带分布口径

| 指标 | 说明 |
|-----|------|
| **价格带** | 0元 / 1.1元 / 3.9元 / 其他（按 `sku_price` 精确匹配） |
| **占比** | 该价格带线索数 / 当月总线索数 × 100% |
| **环比 pp 变化** | 4 月占比 - 3 月占比（百分点） |
| **品类类型** | 正式品 / 孵化品，通过 `category_name` 与 CSV `品类` 匹配后映射 `品类区别` |

### 4.3 资源位×品类类型效率口径

| 指标 | 说明 |
|-----|------|
| **正式品线索数/占比/CVR** | 该资源位下 `cat_type == '正式品'` 的聚合数据 |
| **孵化品线索数/占比/CVR** | 该资源位下 `cat_type == '孵化品'` 的聚合数据 |
| **推荐标签** | 「建议加推孵化品」：孵化品 CVR > 正式品 CVR 且 孵化品占比 < 20% 且 孵化品线索 ≥ 20 |

### 4.4 数据验证基准

| 指标 | 3月 | 4月 | 环比 |
|-----|-----|-----|------|
| 线索数 | 18,066 | 17,492 | -3.18% |
| 首单数 | 2,636 | 1,894 | -28.15% |
| 首单流水 | ¥454.58万 | ¥322.28万 | -29.10% |
| 整体转化率 | 7.30% | 5.41% | -25.89% |
| LTV 均值 | ¥125.97 | ¥92.12 | -26.88% |
| 月活人数 (MAU) | 758,580 | 702,752 | -7.36% |
| 线索生成率 | 2.38% | 2.49% | +4.51% |

### 4.5 关键过滤规则

- **合计行过滤**：`stat_month != '合计'`，必须在所有计算前执行
- **热力图数据门槛**：仅保留 4 月线索数 ≥ 20 的资源位×品类组合
- **品类流量结构 CSV**：过滤 `品类 != '合计'` 及空值行
- **品类名称映射**：Excel `category_name` 与 CSV `品类` 优先精确匹配，未命中则 fallback 到包含匹配，仍未命中标记为「未分类」

## 5. 交互逻辑

### 5.1 月份切换（顶部导航）

三种互斥视图：

#### 3月 / 4月（单月视图）
- **KPI 卡片**：展示所选月份数值；3 月无环比，4 月展示环比
- **趋势图**：仅展示所选月份单线
- **资源位线索占比饼图**：3 月展示 3 月数据，4 月展示 4 月数据
- **品类流量结构柱状图**：3 月展示 3 月正式品/孵化品堆叠，4 月展示 4 月数据 + 环比折线
- **品类流量结构环形图**：3 月展示 3 月占比，4 月展示 4 月占比
- **价格带旭日图**：3 月视图只展示 3 月旭日图，4 月只展示 4 月，对比视图并排展示
- **表格**：通过 CSS `display: none` 隐藏非当前月份列

#### 3-4月对比（默认视图）
- **KPI 卡片**：展示 4 月数值 + 环比
- **趋势图**：双月四线对比（3 月线索/GMV + 4 月线索/GMV）
- **价格带旭日图**：并排展示 3 月和 4 月两个旭日图
- **表格**：展示全部列（3 月 / 4 月 / 环比）

**技术实现**：
- CSS：`body.view-march .col-april { display: none }`
- JS：`updateKPI(month)` 重算 KPI，`renderCharts()` 重绘图表，`setMonth(m)` 切换 body class
- 旭日图在 `renderCharts()` 中根据 `currentMonth` 决定 `chart.clear()` 或 `chart.setOption()`

### 5.2 逐资源位品类明细卡片（折叠/展开）

- 默认折叠，仅显示资源位名称 + 品类数量
- 点击标题展开，展示该资源位下所有品类的线索/CVR/GMV/得分
- 再次点击收起

### 5.3 导出 HTML 月报

- 点击顶部「导出 HTML 月报」按钮
- 将当前视图所有 ECharts 图表转为 PNG 静态图
- 移除交互元素（按钮、导航、折叠状态）
- 生成独立 HTML 文件下载
- 文件命名：`APP线索广告位月报_2026年4月_YYYYMMDD.html`

## 9. 资源位×品类类型效率分析（v3.0 新增）

### 8.1 表格结构

| 列 | 说明 |
|---|------|
| 资源位 | 15 个目标资源位 |

- **「建议加推孵化品」**：当 孵化品 CVR > 正式品 CVR 且 孵化品线索占比 < 20% 且 孵化品线索数 ≥ 20

1. 精确匹配：Excel `category_name` == CSV `品类`
2. 包含匹配：若精确未命中，检查 Excel 名称是否包含于 CSV 名称（或反之）
3. 未分类：仍未命中则 `cat_type = '未分类'`
