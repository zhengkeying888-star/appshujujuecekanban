## 全局设计系统（四屏通用）

### 配色方案：深蓝灰科技风
| 变量 | 色值 | 用途 |
|------|------|------|
| `--bg-base` | `#0f172a` | 页面背景（通透深蓝灰，不是纯黑） |
| `--bg-surface` | `#1e293b` | 卡片表面 |
| `--bg-elevated` | `#334155` | 悬停/高亮卡片 |
| `--border-subtle` | `rgba(148,163,184,0.12)` | 卡片边框（极淡，靠背景层次分隔） |
| `--text-primary` | `#f8fafc` | 主文字（亮白，但不是刺眼的纯白） |
| `--text-secondary` | `#94a3b8` | 副标题、标签 |
| `--text-tertiary` | `#64748b` | 元数据、时间戳 |
| `--accent-primary` | `#3b82f6` | 品牌强调色（亮蓝） |
| `--accent-gradient` | `linear-gradient(135deg, #3b82f6, #8b5cf6)` | 按钮、选中态、头部渐变 |
| `--success` | `#10b981` | 增长/正向 |
| `--danger` | `#ef4444` | 下降/负向 |
| `--warning` | `#f59e0b` | 警告/注意 |
| `--glow-success` | `rgba(16,185,129,0.2)` | 正向卡片发光 |
| `--glow-danger` | `rgba(239,68,68,0.2)` | 负向卡片发光 |

### 字体
- 中文：Noto Sans SC
- 数字/英文：Inter
- 指标数字：40px，font-weight 700，letter-spacing -0.02em，等宽数字

### 卡片规范
- 圆角：16px
- 内边距：24px
- 背景：`--bg-surface`
- 边框：1px solid `--border-subtle`
- 阴影：`0 4px 24px rgba(0,0,0,0.3)`
- 悬停：上移 2px，阴影加深，边框变亮至 `rgba(148,163,184,0.2)`

### 顶部导航栏（固定，64px 高）
- 背景：`--bg-surface` + `backdrop-filter: blur(12px)` + 80% 不透明度
- 底部边框：1px `--border-subtle`
- 左侧：Logo 图标 + 「APP 商业化决策看板」+ 「2026年3-4月」小标签（`--bg-elevated`，圆角 6px）
- 中间：月份切换 Pill — 「3月」|「4月」|「对比分析」
  - 未选中：`--text-secondary`，无背景
  - 选中：白色文字，`--accent-gradient` 背景，圆角 8px，内边距 6px 16px
- 右侧：「导出 HTML 月报」按钮，`--accent-gradient` 背景，白色文字，圆角 8px

## 屏幕1布局：诊断结论区（首屏，30 秒读完）

### 模块 A：核心诊断结论卡（全宽）
- 背景：`--bg-surface`，叠加极淡渐变 `linear-gradient(135deg, rgba(59,130,246,0.06), rgba(139,92,246,0.06))`
- 左上角装饰：4px 宽 `--accent-primary` 竖条
- 内容（从上到下）：
  1. 标签：「诊断结论」（11px，大写，`--accent-primary`，字间距 0.1em）
  2. 主标题：「4月 GMV 环比下降 12.9%（¥-42.5M）」（32px，粗体，`--text-primary`）
  3. 副标题：「主因：完课率下滑 7pp + 客单价下降 8%」（16px，`--text-secondary`）
  4. 进度条：红绿分段条（总宽 100%，高 6px，圆角）
     - 红色段 70%，`--danger`
     - 绿色段 30%，`--success`
     - 下方标注：左「负向驱动因素 (70%)」，右「正向抵消因素 (30%)」（12px，`--text-tertiary`）

### 模块 B：3 个量化信号卡（横排，等宽，间距 20px）
每张卡片：
- 背景：`--bg-surface`
- 左边框：3px solid（恶化 = `--danger`，改善 = `--success`）
- 顶部：指标名（13px，`--text-secondary`，大写）+ 环比标签（圆角 4px，内边距 2px 8px）
  - 标签样式：改善 = `rgba(16,185,129,0.15)` 背景 + `#10b981` 文字；恶化 = `rgba(239,68,68,0.15)` 背景 + `#ef4444` 文字
  - 标签内容：箭头 + 百分比，如「↓ 10.8%」
- 中部：指标数值（40px，粗体，`--text-primary`，等宽字体）
  - 例：「34.2%」「12.5%」「2.49%」
- 底部：较上月变化绝对值（13px，`--text-tertiary`）
  - 例：「vs 3月: -4.1pp」「vs 3月: -2.8pp」「vs 3月: +0.11pp」
- 卡片发光：
  - 恶化卡片：`box-shadow: 0 0 20px var(--glow-danger)`
  - 改善卡片：`box-shadow: 0 0 20px var(--glow-success)`
- 三张卡片内容：
  1. 完课率 34.2% ↓ -41.1%（红色左边框 + 红光）
  2. 首单转化率 12.5% ↓ -16.7%（红色左边框 + 红光）
  3. 线索生成率 2.49% ↑ +4.5%（绿色左边框 + 绿光）

### 模块 C：转化率变动幅度排序图（全宽，高 340px）
- 卡片标题：「转化率变化对比分析」+ 筛选图标
- 图表类型：横向分组柱状图 + 差异连线
- 6 个转化环节，按环比绝对值从大到小排序（恶化最严重的在最上方）：
  - 曝光→点击、点击→领课、领课→好友、好友→到课、到课→完课、完课→首单
- 每组两根柱子并排：
  - 3月柱：`--accent-primary` 60% 不透明度，左侧圆角
  - 4月柱：`--accent-primary` 100% 不透明度，右侧圆角
  - 柱高 16px，组间距 24px
- 差异连线：灰色虚线（`--text-tertiary` 40% 不透明度）连接两根柱子末端
- 差异标签：连线中间显示环比值，红色↓或绿色↑+百分比
- 行背景：恶化行 `rgba(239,68,68,0.06)`，改善行 `rgba(16,185,129,0.06)`
- Y 轴标签：环节名（14px，`--text-primary`）右对齐
- X 轴：百分比 0-100%，每 20% 一条虚线网格（`--border-subtle`）
- Tooltip：浮动卡片，`--bg-elevated`，圆角 12px，显示环节名、3月值、4月值、环比变化
- 动画：柱子从左向右生长，800ms，逐行错开 100ms

### 模块 D：行动优先级速览（转化率图下方，左右两列，间距 20px）
- 左卡片：「产研侧」— `--bg-surface` + 左边框 `--accent-primary`
  - 头部：标签图标 + 标题（16px 粗体）
  - 2 项策略，每项包含：
    - 优先级徽章：P0 = `--danger` 背景白字（圆角 4px），P1 = `--warning` 背景黑字
    - 策略标题（15px，粗体，`--text-primary`）
    - 一句话依据（13px，`--text-secondary`）
- 右卡片：「策略运营侧」— `--bg-surface` + 左边框 `--success`
  - 同上结构
