- 中文：Noto Sans SC
- 数字/英文：Inter
- 指标数字：40px，font-weight 700，等宽数字

### 卡片规范
- 圆角：16px，内边距：24px
- 背景：`--bg-surface`，边框：1px solid `--border-subtle`
- 阴影：`0 4px 24px rgba(0,0,0,0.3)`
- 悬停：上移 2px，阴影加深，边框变亮

### 顶部导航栏（固定，64px 高）
- 背景：`--bg-surface` + `backdrop-filter: blur(12px)` + 80% 不透明度
- 左侧：Logo + 「APP 商业化决策看板」+ 「2026年3-4月」标签
- 中间：月份切换 Pill — 「3月」|「4月」|「对比分析」（选中态为 `--accent-gradient` 背景）
- 右侧：「导出 HTML 月报」按钮

## 屏幕2布局：转化链路全景区

### 整体布局：左右分栏（左 55% / 右 45%），间距 20px，下方全宽诊断卡片区

### 左侧大卡片：转化漏斗 + 下钻链路
- 卡片标题：「转化链路全景」+ 地图/漏斗图标
- 占据卡片 60% 高度的核心漏斗：
  - ECharts funnel，3 层：曝光 UV → 点击 UV → 线索数
  - 每层用渐变色填充：
    - 曝光层：`#1e3a5f` → `#3b82f6`（深蓝到亮蓝）
    - 点击层：`#312e5a` → `#8b5cf6`（深紫到亮紫）
    - 领课层：`#1e3a3a` → `#10b981`（深绿到翠绿）
  - 标签 inside：白色文字，显示「环节名 + 人数 + 转化率」
  - 3月/4月切换：Toggle Switch 样式（同导航栏 Pill），切换时漏斗层平滑过渡
- 占据卡片 40% 高度的下钻链路（漏斗下方）：
  - 横向流程图，4 个节点：好友 → 到课 → 完课 → 首单
  - 节点样式：圆角矩形，`--bg-elevated` 背景，1px `--border-subtle` 边框
    - 中心圆：转化率%（14px，粗体，`--accent-primary`）
    - 下方：环节名（12px，`--text-secondary`）
    - 下方：人数（13px，`--text-primary`）
  - 节点大小 ∝ 人数规模（宽 64-96px）
  - 边线：2px，`--text-tertiary`，带箭头，线上标注环节间转化率（11px）
  - 悬停节点：边框高亮为 `--accent-primary`，带发光效果

### 右侧卡片：各阶段转化率对比
- 卡片标题：「各阶段转化率对比」+ 对比图标
- ECharts 横向分组柱状图（同屏幕1风格）
- 6 个环节按链路顺序排列（不排序）：
  - CTR、领课转化率、好友率、到课率、完课率、首单转化率
- 每组两根柱子：3月（`--accent-primary` 60% 不透明度）+ 4月（`--accent-primary` 100%）
- 柱旁标注环比：红色↓或绿色↑+百分比
- 恶化最严重的整行高亮 `rgba(239,68,68,0.08)` 背景
- Y 轴：环节名，X 轴：百分比
- 底部图例：3月/4月 圆点

### 下方全宽：诊断卡片区（只展示变化最大的 3 张）
- 3 张卡片横排，间距 20px
- 每张卡片：`--bg-surface`，圆角 16px
  - 顶部左：环节名（16px，粗体，`--text-primary`）
  - 顶部右：责任方标签（圆角 4px，`--bg-elevated`，`--text-secondary`）
  - 中部：3月值 → 箭头 → 4月值（20px，粗体）+ 环比（18px，红/绿）
  - 底部：一句话问题描述（14px，`--text-secondary`）+ 图标
- 左边框 3px：恶化 = `--danger`，改善 = `--success`
- 内容示例：
  1. 「完课率：58.0% → 34.2% ↓41%」— 「完课率大幅下滑，督学机制需复盘」
  2. 「首单转化率：15.0% → 12.5% ↓17%」— 「高价课转化走弱，需优化价格策略」
  3. 「线索生成率：2.38% → 2.49% ↑4.5%」— 「MAU 下降但线索生成率提升，流量质量改善」
